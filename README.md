# ATS Resume Keyword Analyzer

A professional Resume Keyword Analyzer built using Python and Streamlit.

## Features
- ATS Match Score
- Resume Keyword Extraction
- Missing Keyword Detection
- Suggestions to Improve Resume
- Modern UI
- PDF Resume Upload

## Technologies Used
- Python
- Streamlit
- NLP
- Scikit-learn
- PyPDF2
- SQLite

## Run Project

```bash
python -m streamlit run app.py
